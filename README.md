# Portfólio Power BI
Coleção de dashboards desenvolvidos em Power BI para análise de dados e geração de insights.

## Projetos

| Projeto | Descrição |
|----------|------------|
| [Dashboard de Vendas](./sales-dashboard) | Análise de faturamento, volume vendido, marcas e distribuição geográfica. |
| [Dashboard de Produção](./Manufacturing_Dashboard) | Monitoramento de indicadores operacionais, incluindo produção aprovada, rejeições, horas produtivas e eficiência do processo. |
| [Dashboard de RH](./HR_Dashboard) | Análise de indicadores de Recursos Humanos, com acompanhamento de contratações, desligamentos, turnover e perfil dos colaboradores. |
