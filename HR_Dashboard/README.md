# Dashboard de RH
Dashboard desenvolvido para monitoramento de indicadores de recursos humanos, permitindo acompanhar contratações, desligamentos, turnover e perfil dos colaboradores.

## Principais análises
- Total de colaboradores
- Contratações
- Demissões
- Taxa de turnover
- Distribuição por cidade
- Distribuição por gênero
- Colaboradores por área e cargo
